	\PARstart{C}{ette} section donne une introduction générale sur le problème scientifique abordé et décrit la structure de l'article. Des questions souvent abordés ici sont:
	\begin{itemize}
	\item Quels sont les applications du problème abordé?

	Cette section permets de décrire l'état de l'art concernant la question abordé (c-à-d les meilleurs solutions disponibles à présent) et de positionner votre travail par rapport à cet état de l'art.
	Les différents articles que vous auriez lu et utilisé devraient être correctement référencés (Example: \cite{small},\cite{big}).
	Les informations bibliographiques sur les devraient être encodés dans le fichier \texttt{References.bib} avec la syntaxe indiqué par les exemples.
	Articles publiés sur une revue scientifique / dans les conference proceedings, ainsi que des livres, sont des examples de bonnes références.
	Par contre, la citation de sources web doit être limité le plus possible (permis dans le cas de la documentation d'outil informatiques). Plus précisément, Wikipedia \emph{n'est pas} un source approprié pour un rapport scientifique.
